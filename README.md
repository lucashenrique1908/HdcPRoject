HdcPRoject

This repository contains the HdcPRoject, a static webpage created as part of front-end development practice.

Live version:
https://lucashenrique1908.github.io/HdcPRoject/

Description

HdcPRoject is a simple, structured webpage that presents basic content using HTML and CSS.
The project serves as practice for building layouts and organizing visual elements on a static page.

Technologies Used

 - HTML

 - CSS

Purpose

The main objectives of this project are to:

* Practice semantic HTML structure

* Apply CSS styling and layout techniques

* Build a basic static webpage with organized visual content

This project was developed for training and skill development, focusing on foundational front-end techniques.

How to Run Locally:
